# Third-party notices for DocumentEditor assets

This folder contains locally packaged JavaScript assets used by `PhialeTech.DocumentEditor`.

Included libraries:

- `@tiptap/core`
  - license: MIT
  - usage: headless document editor runtime
- `@tiptap/starter-kit`
  - license: MIT
  - usage: core paragraph, heading, list, quote, rule, undo, redo commands
- `@tiptap/markdown`
  - license: MIT
  - usage: Markdown import and export
- `@tiptap/extension-*`
  - license: MIT
  - usage: underline, color, highlight, text-style, text-align, link, image, table, placeholder, character-count, font-family

This implementation intentionally uses the open-source `@tiptap/markdown` package and does not use paid Tiptap Conversion or Import features.
