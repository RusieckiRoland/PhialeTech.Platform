# PhialeGis.Library

`PhialeGis.Library` is a proprietary product repository created and maintained by Roland Rusiecki.

This repository is used to develop the `PhialeGis` GIS application for .NET, centered around a domain-specific language for GIS commands, workflows, and user interaction patterns. It also serves as the foundation for an evolving design system and the `PhialeTech` component palette, intended both for GIS applications and for other professional desktop software.

The codebase is being developed for multiple .NET UI platforms:

- `WPF`
- `WinUI 3`
- `Avalonia`

## Scope

The repository currently covers the following primary areas:

- Development of the `PhialeGis` GIS application and its supporting architecture.
- Evolution of a DSL-based application model for GIS commands, document flows, and user interaction patterns.
- Development of a cross-platform design system for future products.
- Development of the `PhialeTech` component suite for GIS and other professional desktop applications targeting `WPF`, `WinUI 3`, and `Avalonia`.
